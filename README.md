<h2 align="center"> 👨‍💻 What I Do </h2>

- Build AI agents that run experiments, write code, evaluate results, and improve ML systems.
- Design benchmarks and evaluation infrastructure for machine learning engineering tasks.
- Optimize compact language models with quantization, test-time training, recurrence, compression, and search.
- Build backend services and developer tools with Python, Rust, Go, and TypeScript, then deploy them with Kubernetes and cloud infrastructure.
- Turn research ideas into working systems, reproducible workflows, and open-source releases.

- **[Hyperledger Fabric Python SDK](https://github.com/hyperledger/fabric-sdk-py)**: I maintained the SDK and mentored contributors.

<h2 align="center"> 🛠 Technical Range </h2>
